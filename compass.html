Month Compass

Widerstand — Creation Month (supported by Stabilization)

North Star: Create clarity through one creative decision. Start Signal: Send producer update message → first gentle DaVinci session (30–60 min).

1) Keystone Outcome (end of month)

  • Full second cut exists (DaVinci Resolve).
  • Decision made on 1 version (max 2 if truly necessary).
  • Decision communicated to the Producer (Filmwerkstatt) so sound work can proceed focused.

Rule: Don’t change the goal. Change the approach (place, time, session type) if needed.

2) Weekly Anchors (repeat every week)

Daily (Stabilization)

  • Yoga: 15–30 min (Yoga with Adriene playlist)
  • Meditation: morning 5 min + evening ≥10 min
  • Smoothie / protein shake as supportive nutrition (no perfection)

Movement

  • Dance workouts: Weeks 1–2 → 2×/week (30–60 min)
  • Dance workouts: Weeks 3–4 → 3–4×/week (30–60 min)
  • Face yoga: Week 1 → 2× (optional) → increase gently if nourishing

Creation

  • Widerstand blocks: 4–5 days/week
  • Session length: 30 min – 3 hours (stop before exhaustion)
  • One block/day max (choose a block type below)

Life & Relationships

  • Boyfriend: protected weekend block (presence, not multitasking)
  • Friends: 1–2 light contacts/week (signal care)
  • Perspective Trip: 1×/week (friend visit OR nature walk OR museum/train ride)

Awareness / Community

  • Awareness program: 1 intentional input/week (live at 5pm or recording)
  • Sunday live webinar: optional, no guilt
  • Creative collective Sundays: Create session (work/journal) or Masterclass (learn)

3) Widerstand Session Menu (pick ONE per session)

  • Technical exploration: learn/test DaVinci workflow (short tutorial, settings, imports)
  • Edit work: advance the second cut (structure, pacing, continuity)
  • Sound work: test-mix one key scene (light experiments only)
  • Review & notes: watch as viewer, mark decisions, write “next time…”

Minimum viable session: 30 minutes. Leave a thread: write one line “Next time I’ll …”.

4) Life Admin Blocks (LAB) — protected container

  • Frequency: 2–3×/week
  • Duration: 45–90 min (hard stop)
  • Examples: therapy calls/follow-ups, time-sensitive documents/forms, necessary emails
  • Recovery required: after each LAB do a downshift (walk / shower+music / face yoga / rest)

If the week feels full: reduce creative volume, not admin integrity (missed calls create lingering stress).

5) Studio Decision (micro-project boundaries)

  • This month: appointment → decision (yes/no/yes-with-conditions) → minimal setup to work
  • Not this month: perfecting the space, full move, identity project

Decision prompts

  1. Can I work on Widerstand here within 7 days?
  2. Does shared setup reduce or increase mental load?
  3. Is cost tolerable for the next 3 months (not forever)?
  4. Does it simplify creation?

6) Weekly Reset (10 minutes)

  1. What worked last week?
  2. What drained me unnecessarily?
  3. What’s my one creative priority this week?
  4. Where do I place: boyfriend block, perspective trip, 2–3 admin blocks?

No full rescheduling. Gentle steering only.

7) “Not This Month” (protect focus)

  • No major relocation decisions
  • No studio perfection
  • No social over-commitment
  • No backlog/catching up mindset
  • No guilt about live webinars (recordings fully count)

8) Motivation Placement

Minimalism documentary (watch once, intentional):

  • Place it in Week 1, mid-week evening.
  • After yoga + meditation. No work afterwards.
  • Journal 10 minutes: “What can be simpler this month?”

Link: theminimalists.com/films

Principle: Focus is selective presence. Stable anchors protect adventure. One month at a time.
